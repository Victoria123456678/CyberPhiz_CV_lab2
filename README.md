Что сделано
Запущен Ollama сервер
Использована модель qwen2.5:0.5b
Реализован скрипт для запросов к LLM

Как запустить
1. Установить Ollama
2. Скачать модель:
ollama pull qwen2.5:0.5b
3. Запустить сервер:
ollama serve
4. Запустить скрипт:
python main.py
